## exp_1^2numberguessing.c
实验1作业 猜数字游戏
用计算机产生一个100以内的随机数，游戏者有6次猜数的机会，猜中即胜，猜不中，提示是猜大了还是猜小了，继续猜，直至猜满6次为止。游戏结束后给出评语。
